# QCF Uganda — Cash for Climate Adaptation

## How to reproduce

### Requirements
- Python 3.10+ with: `pandas`, `numpy`, `matplotlib`, `scipy`

### Run
```
cd code
python qcf_analysis.py
```
This generates all 10 figures in `drafts/figures/` from `raw/villages_clean.csv`.

## Folder structure

```
raw/
  villages_clean.csv        Input data (134 settlements with coordinates + population estimates)
code/
  qcf_analysis.py           Single script: clusters, randomization, power calculations
drafts/
  QCF_Uganda.pdf            Paper draft
  figures/                  Figures 1-10 (generated by qcf_analysis.py)
```
